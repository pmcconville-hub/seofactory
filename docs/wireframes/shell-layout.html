Sidebar Levels

The sidebar supports 3 navigation levels that change based on context:

Level 1 (no project): Projects, Settings, Admin

Level 2 (project selected): Back, Maps list, Project Settings

Level 3 (map selected): Dashboard, Setup Wizard, Topics, Audit, Insights, Gap Analysis, Quality, Planning, Calendar, KP Strategy, Entity Authority, Entity Health

Collapsible Sidebar

Click the ‹ button on the sidebar edge to collapse it from 240px to 56px. In collapsed mode, only icons are shown. Hover the collapse button to expand temporarily.

Breadcrumb Bar

The breadcrumb bar sits between the sidebar and main content at the top. It shows the navigation path and updates as the user drills deeper: Projects > My Site > Main Map > Dashboard.
